# Alpha Tester Guide

Release: Closed Alpha 0.1.0-alpha.1

Duracion sugerida de prueba: 7 dias.

## Que probar

- Inventario: agregar, editar, eliminar y marcar productos abiertos.
- Compras: registrar compra rapida y revisar que actualice el inventario.
- Consumo: revisar resumen de gasto, categorias y tendencias.
- Asistente: hacer preguntas sobre stock, compras y alertas.
- Smart Input: escribir una compra en lenguaje natural y confirmar el resultado.
- Ajustes: editar datos del hogar, integrantes, presupuesto y preferencias.
- Product Intelligence: revisar feedback, exportar feedback y refrescar metricas.
- PWA: instalar la app y abrirla desde el icono.

## Tareas sugeridas

1. Agrega 5 productos reales de tu hogar.
2. Registra 2 compras, una manual y una con Smart Input.
3. Marca al menos 1 producto como abierto.
4. Cambia el presupuesto mensual y revisa Consumo.
5. Haz 3 preguntas al Asistente.
6. Envia feedback desde Ajustes.
7. Exporta feedback desde Product Intelligence.
8. Cierra y vuelve a abrir el navegador para confirmar persistencia local.

## Como enviar feedback

Entra a Ajustes, abre la seccion Feedback y completa:

- Tipo.
- Prioridad.
- Descripcion.
- Que esperabas.
- Logs opcionales si viste un error.

Adjuntar captura estara disponible proximamente.

## Como instalar como PWA

En mobile:

- Abre la URL de la Alpha en el navegador.
- Usa el menu del navegador.
- Elige "Agregar a pantalla de inicio" o "Instalar app".
- Abre homemAId desde el icono creado.

En desktop:

- Abre la URL de la Alpha.
- Si el navegador muestra el icono de instalar, usalo.
- Si no aparece, usa el menu del navegador y busca la opcion de instalar app.

## Como reportar errores

Incluye:

- Pantalla donde ocurrio.
- Pasos para reproducir.
- Resultado esperado.
- Resultado obtenido.
- Navegador, dispositivo y sistema operativo.
- Si paso despues de borrar datos, usar modo incognito o instalar como PWA.

## Nota de privacidad

Esta Alpha usa almacenamiento local del navegador. Si borras datos del navegador puedes perder la informacion registrada.

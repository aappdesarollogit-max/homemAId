# Feedback Process

## Captura

Durante Alpha Cerrada, el usuario envia feedback desde Ajustes en la seccion `Feedback`.

Campos:

- tipo
- descripcion
- que esperaba
- prioridad
- captura placeholder
- logs opcionales

## Clasificacion

Los tipos soportados son:

- BUG
- MEJORA
- IDEA
- CONFUSION
- PREGUNTA
- PERFORMANCE
- UX
- OTRO

## Estados

- Nuevo
- En revision
- Aceptado
- Rechazado
- En desarrollo
- Resuelto
- Liberado

## Flujo operativo

```text
Nuevo -> En revision -> Aceptado/Rechazado -> En desarrollo -> Resuelto -> Liberado
```

## Exportacion

El dashboard de producto permite exportar feedback en JSON o CSV para revision externa, analisis con stakeholders o carga futura en herramientas de producto.

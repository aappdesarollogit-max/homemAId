# Mobile Testing Checklist

## iPhone Safari

- Abrir `/`.
- Abrir `/login`.
- Abrir `/registro`.
- Abrir `/dashboard`.
- Validar que no exista scroll horizontal.
- Validar safe area inferior con bottom navigation.

## Android Chrome

- Abrir dashboard.
- Probar navegacion inferior.
- Validar inputs y botones con tap targets comodos.
- Validar scroll interno en formularios largos.

## Desktop Responsive

- Revisar desktop ancho.
- Revisar tablet.
- Revisar mobile emulado.
- Confirmar que sidebar desktop sigue visible solo en desktop.

## Standalone

- Instalar como PWA.
- Abrir desde icono.
- Confirmar `start_url` en `/dashboard`.
- Revisar que bottom nav no quede tapada.

## Navegacion

- Inicio.
- Inventario.
- Compras.
- Consumo.
- Asistente.
- Ajustes.

## Formularios

- Crear producto.
- Editar producto.
- Registrar apertura.
- Crear compra con varios productos.
- Editar ajustes del hogar.
- Agregar integrante.
- Configurar preferencias.

## Scroll

- Chat con muchos mensajes.
- Lista de inventario larga.
- Historial de compras.
- Formularios largos en mobile.

## LocalStorage

- Crear producto y recargar.
- Crear compra y recargar.
- Cambiar ajustes y recargar.
- Verificar que los datos persisten.

## Reset Demo

- Restaurar ajustes.
- Confirmar que inventario/compras no se pierden por cambios de ajustes.

## Asistente IA Local

- Preguntar productos criticos.
- Preguntar gasto mensual.
- Preguntar resumen del hogar.
- Usar sugerencias rapidas.
- Limpiar chat.

## Consumo

- Registrar compra.
- Ver cambio en gasto mensual.
- Ver tiendas, categorias y top productos.

## Compras

- Registrar compra con producto existente.
- Registrar compra con producto nuevo.
- Eliminar compra.

## Inventario

- Buscar.
- Filtrar.
- Crear.
- Editar.
- Eliminar.
